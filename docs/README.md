# Математические формулы

## Нахождение площади
- Rectangle: a * b
- Triangle: a * h
- Circle: Pi * r * r
- Square: a * a

## Нахождение периметра
- Rectangle: 2 * (a + b)
- Triangle: a + b + c
- Circle: 2 * Pi * r
- Square: 4 * a
